# Face Authentication System

## Quick Start Guide

### 1. Setup
Runs these commands in your terminal to install the necessary libraries:

```bash
pip install -r requirements.txt
```

### 2. Run the App
Start the system:

```bash
python auth_system.py
```

### 3. How to Use
Once the app starts, look at the terminal:

*   **Type `E` to Enroll**:
    1.  Enter your Name.
    2.  Press **SPACE** to take 3 photos (Front, Left, Right).
    
*   **Type `A` to Authenticate**:
    1.  Select **'C'** for Camera.
    2.  Follow the instructions on screen (Blink, Smile, Turn Head).
